#Algorithm (Python)

프로젝트 개요
대학교 소프트웨어 전공 수업의 일환으로,
Python을 활용해 기본적인 알고리즘과 자료구조를 학습하고
수업 과제 및 실습 내용을 코드로 정리한 저장소입니다.
알고리즘의 동작 원리와 문제 해결 과정을 이해하는 것을 목표로 진행했습니다.

학습 내용
- DFS / BFS 기반 탐색 알고리즘
- 정렬 알고리즘 (기본 정렬, 비교 기반 정렬)
- 탐색 알고리즘 (선형 탐색, 이진 탐색)
- 재귀 호출 구조 및 종료 조건 설계
- 리스트, 스택, 큐 등 기본 자료구조 활용

구현 방식
- Python을 활용한 알고리즘 직접 구현
- 수업에서 다룬 문제를 중심으로 핵심 로직 단위로 코드 작성
- 실행 결과를 통해 알고리즘 동작 흐름 확인
- 주석을 통해 문제 접근 방식과 풀이 과정 정리

학습 성과 및 배운 점
알고리즘을 결과 위주로 풀이하기보다,
문제를 분해하고 조건에 따라 흐름을 설계하는 사고 방식을 학습했습니다.
이를 통해 새로운 문제에서도 구조를 빠르게 파악하고,
적절한 알고리즘을 선택해 적용하는 기본기를 다질 수 있었습니다.
